Low poly zombies.
- Delete Demo folder if you do not need it anymore.
- Fat zombie has its own animations for run, attack and idle. For death you can use common animation.
- For animation preview set Zombie 1 model instead of default unity one.
- You can use Jump_Idle and Jump_Landing for falling and landing.
- Vertex color shader - https://forum.unity3d.com/threads/standard-shader-with-vertex-colors.316529/
